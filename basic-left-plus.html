Adding the "Plus"

Astro "plus" versions use modern JavaScript API's that aren't supported by older browsers. Include astro-plus.js in your HTML, and add this short snippet of code in the <head> of your HTML document to feature test:

<script>
    if ( 'querySelector' in document && 'addEventListener' in window ) {
        document.documentElement.className += 'js';
    }
</script>

Older and less capable browsers will see the basic version. Modern browsers (including IE 9 and higher) will get the "plus" version.
