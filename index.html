Getting Started

Astro is a collection of five mobile-first navigation patterns. The "plus" version of each pattern includes an expand-and-collapse menu on small screens.

  • Basic | Basic Plus
    A centered navigation and logo.
  • Basic Left | Basic Left Plus
    A left-aligned navigation and logo.
  • Navbar | Navbar Plus
    An inline navigation and logo.
  • Navbar Left | Navbar Left Plus
    An inline navigation and logo with left-aligned navigation.
  • Stacked | Stacked Plus
    Stacked navigation and logo.

Using Multiple Patterns

For simpicity, each navigation pattern uses the same naming conventions. If you will be using more than one pattern on a project, you'll need to change the class names to avoid conflicts.

For example, if you were combining Astro Basic and Astro Stacked, you might rename .nav to .nav-basic and .nav-stacked, respectively.

Active Link Styling

There's a placeholder in the CSS to add styling to the current page in the navigation menu.

/*  Placeholder for active link styling */
/*  .nav > li.active > a { */
/*      Add your styles here */
/*  } */

Note: If you're using WordPress, check out this great tutorial by Todd Motto on how to add the .active class using a simple PHP function.

Browser Support

Astro works in all modern browsers, and IE 8 and above.

Because IE 8 does not recognize media queries, it will receive the small-screen navigation option even on big screens (learn more about mobile-first and IE). The "plus" versions of Astro require modern JavaScript API's that are supported in IE 9 and later. IE 8 will instead get the basic navigation option.
