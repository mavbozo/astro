How to use Astro Basic

1. Include Astro on your site.

<link rel="stylesheet" href="astro-basic.css">

For the basic versions of Astro, you only need to include the CSS file, not the JavaScript.

2. Add the markup to your HTML.

<nav class="nav-wrap">
    <a class="logo" href="#">My Brand</a>
    <ul class="nav">
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
    </ul>
</nav>

3. Pat yourself on the back.

Seriously, you're done. Nice work! Modify as needed.
